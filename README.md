# Project: The 8th Protocol (The8th)
## Architect: backstage8
## Tech Stack: ICP (Internet Computer), IPFS, Motoko, Web3 WebApp.

### Core Logic Manifest (IMMUTABLE):
1. **Fiscal Policy:** A 20% flat tax is applied to ALL in-game transactions, routed to the Architect's Sovereign Wallet.
2. **Geography:** 7 Visible Continents + 1 Hidden High-Tech Continent (The 8th). 70 Cities total.
3. **Time Scale:** 1 Real Month = 1 In-Game Year.
4. **Progression:** 4 Classes (Swordsman, Fighter, Ranger, Blacksmith). Tier upgrade every 20 levels via hidden "Ascension Tasks".
5. **Economy:** All assets (Characters, Mounts, Corpses) are NFTs.
6. **Architect Authority:** Full visibility of hidden player data, instant teleportation, and climate control via Developer ID.

### Development Goal:
Build a dark-gothic realistic web app using AI-driven automation and decentralized hosting.
